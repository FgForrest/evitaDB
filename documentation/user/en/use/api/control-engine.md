    The evitaDB database engine can be fully controlled programmatically through its Control API. This API allows developers to manage catalogs, monitor the engine's health, and perform various administrative tasks directly from their applications. Our web console evitaLab uses this API to for all its management functionalities. This document provides an overview of all supported engine level operations available also to you.
